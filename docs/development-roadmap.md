# Development Roadmap

## Phase 1: Foundation

- Monorepo structure
- Backend app, security middleware, error handling, database connection
- Mongoose schemas and indexes
- Frontend routing, layout, auth store, API client
- Public pages and dashboard shells

## Phase 2: Authentication

- Candidate registration/login/logout
- Admin login and role authorization
- Refresh token rotation
- Email verification and password reset
- Protected frontend routes

## Phase 3: Public CMS

- Officer CRUD with image upload
- News, notices, gallery, and contact messages
- Public listing/detail pages
- Admin tables and forms

## Phase 4: Recruitment

- Job circular CRUD
- Candidate profile, education, experience, and document upload
- Job application workflow
- Application review, status updates, shortlist/approve/reject

## Phase 5: Payment and Admit Cards

- SSLCommerz initialization and validation
- bKash/Nagad support abstraction
- Payment history
- Roll number generation
- PDFKit admit card with QRCode

## Phase 6: Production

- Logging, monitoring, and backups
- Deployment docs for Vercel, Render, and MongoDB Atlas
- Security audit and rate-limit tuning
- Full test coverage for critical flows
